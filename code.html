04:15

Para el dorado, es fundamental la temperatura. La madera respira, se expande y contrae con el calor del taller.

04:20

04:25

Si hierve, perderá su fuerza adhesiva. Se volverá quebradiza y el oro no se fijará como debe sobre la superficie preparada.

04:32

Observad la consistencia. Debe fluir como miel caliente, ni muy líquida ni muy espesa. Es el punto exacto que buscamos.

04:45

05:10
